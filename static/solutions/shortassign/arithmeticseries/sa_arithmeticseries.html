Problem 2: Arithmetic series

When you sum the integers from 1 through n, that sum is called an arithmetic series. For example, when n = 4, the sum is 1 + 2 + 3 + 4, which equals 10. There's a nice formula for the arithmetic series, which you might have seen:

$1 + 2 + \cdots + n = \displaystyle\frac{n(n+1)}{2} \ .$

Your job is to write a Python function that takes a positive integer parameter n, computes the sum 1 + 2 + ⋯ + n directly, compares the computed sum with the value given by the above formula, and returns a boolean indicating whether the sum equals the formula's value. This function should take no input from the console, nor should it print any output to the console.

To compute the sum directly, you are required to use a for-loop, and the header of the for-loop must make an appropriate call to the built-in function range. Things to note:

  • Remember that the last number in the list returned by range is 1 less than the value of the last parameter you give to range.
  • You'll want to have a variable that accumulates the sum of the numbers in the list returned by range. You might be tempted to name this variable sum, which would be a perfectly good and descriptive name for this variable. But you should not name this variable sum, because Python has a built-in function with this name. If you use the name sum, you'll see the little yellow warning triangle in Eclipse, and this time you should heed the warning. So use a variable name that describes the purpose of the variable, but that is not sum.
  • Remember that the operator == compares two values and gives a boolean that tells you whether the two values are equal. Your function returns a boolean that is the result of this comparison. Therefore, your function should not need to include an if-statement anywhere, and if it does, then you will not get a 2 on this assignment.

I was able to write this function so that its body contained just four non-blank, non-comment lines.

In addition to the function that checks the arithmetic series, you should write code that repeatedly prompts the user for a number, calls the function with that number, and prints out to the console a message indicating whether the formula works for that number. Your program should terminate once a negative number or 0 is entered. Here's output from my program:

Enter a number: 5

Remember that you can call the raw_input function to get a string from the console, and then you can call int on the result of raw_input to convert the characters entered into an int. Don't worry about error checking, so that if the input is five, then yeah, an error occurs. A reminder that if you supply a string as the parameter to raw_input, then that string will be the prompt to the user. Do not call your function that checks the arithemetic series on a negative number or 0. In order to make sure that your function is called only on positive numbers, you are allowed to use an if-statement in the part of your program that gets input and calls your function. You may also use an if-statement when deciding what output to print to the console.
